# Texas Statutes Legibility Chrome Extension

This Chrome extension improves the readability of Texas statutes pages at https://statutes.capitol.texas.gov/Docs by applying better typography and a dark theme.

## Features

- Dark theme with improved contrast
- Larger, more readable fonts (18px)
- Better font selection (Georgia serif)
- Improved spacing and line height
- Enhanced link styling
- Responsive design with centered content

## Installation

1. Open Chrome and navigate to `chrome://extensions/`
2. Enable "Developer mode" in the top right
3. Click "Load unpacked" and select the `legibility-extension` folder
4. The extension will automatically apply to Texas statutes pages

## Usage

Simply visit any page under https://statutes.capitol.texas.gov/Docs/ and the improved styling will be automatically applied.

## Files

- `manifest.json` - Extension configuration
- `styles.css` - CSS styles that improve readability
- `icon16.png`, `icon48.png`, `icon128.png` - Extension icons
