Privacy Policy

Google Verbatim Search Chrome Extension

Effective date: March 26, 2026

Summary

Google Verbatim Search does not collect, store, transmit, or share any user data. It has no analytics, no tracking, and no remote servers.

What the extension does

The extension modifies Google search URLs in your browser by adding three URL parameters (tbs=li:1, pws=0, filter=0) to enforce verbatim search results. All processing happens locally in your browser. No data leaves your device.

Permissions used

  • webNavigation — Used to detect when you navigate to a Google search page so the extension can modify the URL parameters before the page loads.
  • tabs — Used to update the current tab's URL with the modified search parameters.
  • Host permission (www.google.com) — Limits the extension's activity exclusively to Google search pages.

Data collection

This extension collects no data of any kind. Specifically:

  • No personal information is collected
  • No browsing history is recorded
  • No search queries are stored or transmitted
  • No cookies are set or read
  • No network requests are made to any server
  • No analytics or telemetry of any kind

Third parties

This extension does not share data with any third party because it does not collect any data.

Open source

The complete source code is available at github.com/markwbennett/Chrome-Scripts.

Contact

If you have questions about this privacy policy, you can open an issue on the GitHub repository linked above.

Changes

If this privacy policy changes, the updated version will be posted at this URL. The extension itself contains no mechanism to notify you of policy changes because it does not communicate with any server.
